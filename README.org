* Building locally
  - Run make from the root directory:
    #+BEGIN_SRC shell -n
      make
    #+END_SRC

  - Use Python's =http.server= module to locally render the pages:
    #+BEGIN_SRC shell -n
      # Python >= 3.7
      python -m http.server --directory=public
    #+END_SRC

  - Open http://localhost:8000/ in the Web browser
